WHAT IS HADOOP?

Hadoop is a framework that is used for batch or offline processing through a web of diffrent devices and computers to form a clustr. Hadoop is a software to handle big data and organize them more quickly and effeciently

WHAT ARE THE COMPONENTS OF HADOOP?

There are 3 components of Hadoop which are HADOOP DISTRIBUTED FILE SYSTEM (HDFS), MAP REDUCE, YARN

HADOOP DISTRIBUTED FILE SYSTEM (HDFS)

The use of HDFS is to store chunks of data into different storages in order for it to become effecient and does not need very large capacity to store them this method is called the Hadoop cluster. It is designed to work on commodity Gardware Devices, which are easy to find and are not expensive that works on HDFS HDFS can handle very large size data with high velocity and variety, making itt effecient and reliable. It is also to access to all components. Each data block can handle 128mb of data bat can also be configurable

MAP REDUCE

Is a programming model used to distribute the processing in parallel in a Hadoop Cluster. When dealing with data, Map Reduce uses 2 phases Map Task and Reduce Task

Map Task

Map Tasks, takes the gathered data and counts the data and analyses the value of each data, this way it shows the values of all data that has been gathered.

Map Reduce

If there are different kinds of data, what Map Task does is it organizes the data into its same category arranging the data into groups where they are the same or in category.

YARN

YARN allows different data processing engines like graph processing, interactive processing, stream processing as well as batch processing to run and process data stored in HDFS (Hadoop Distributed File System) thus making the system much more efficient. Through its various components, it can dynamically allocate various resources and schedule the application processing. For large volume data processing, it is quite necessary to manage the available resources properly so that every application can leverage them.
